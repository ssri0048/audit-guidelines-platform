🔗 GitHub Auto-Commit — บันทึกความรู้ใหม่อัตโนมัติ

เมื่อ AI เรียนรู้ข้อมูลใหม่และผ่าน 6 Quality Gates แล้ว ระบบจะ commit knowledge_store.json ขึ้น GitHub โดยอัตโนมัติ → Netlify deploy ทันที → ทุกคนเห็นข้อมูลใหม่

🤖 AI Research → ✅ 6 Gates → 💾 mergeIntoStore → 🚀 GitHub Commit → 🌐 Netlify Live
GitHub Personal Access Token:
🔑 Token ต้องมีสิทธิ์ repo (Contents: Read & Write)
📖 วิธีสร้าง GitHub Token
  1. ไปที่ github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens
  2. คลิก Generate new token
  3. Repository access: เลือก ssri0048/audit-platform
  4. Permissions → Contents: Read and Write
  5. คลิก Generate token แล้วคัดลอก token มาวางด้านบน

⚠️ Token เก็บใน IndexedDB ของ browser นี้เท่านั้น — ไม่ได้ส่งออกนอกนอกจากไปที่ api.github.com

🚀 Netlify Deploy Hook URL:
🌐 เมื่อ admin commit ข้อมูลใหม่ → Netlify redeploy อัตโนมัติ (~1-2 นาที) → ทุกคนเห็นข้อมูลใหม่โดยไม่ต้อง token
📖 วิธีสร้าง Netlify Deploy Hook
  1. ไปที่ app.netlify.com → Site → Site configuration → Build & deploy → Build hooks
  2. คลิก Add build hook → ตั้งชื่อ "audit-sync" → Save
  3. คัดลอก URL ที่ได้ มาวางด้านบน
