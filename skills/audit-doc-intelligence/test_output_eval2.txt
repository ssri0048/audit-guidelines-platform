New type key justification:
  'supply_chain_risk' is a new type key. The closest existing type is 'risk_management'
  (ERM/COSO/ISO 31000), but supply chain / TPRM is a sufficiently distinct domain
  with its own standards (ISO 28001, WCO SAFE, CTPAT), regulatory landscape, and
  audit procedures. A dedicated type key enables precise document detection and
  prevents false positives against general enterprise risk documents.
